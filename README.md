# SEA-CROGS — DOE ASCR PI Meeting 2026

Presentation materials for the SEA-CROGS project at the DOE ASCR Principal Investigator
Meeting, 2026. SEA-CROGS is *Scalable, Efficient and Accelerated Causal Reasoning
Operators, Graphs and Spikes for Earth and Embedded Systems*.

## Overview talk (15 min)

[`trask_PImeeting_2026.pptx`](trask_PImeeting_2026.pptx) — project overview,
N. Trask and G. Karniadakis.
